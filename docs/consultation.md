The following relationships are associated with this resource and can be retrieved by attaching a `relationships` array as a query param:

| Resource            | Description                                                        |
| ------------------- | ------------------------------------------------------------------ |
| `absent`            | Users who were/are absent from the consultation                    |
| `attendance`        | Attendance lsit showing when each user joined and left the meeting |
| `confirmedPayments` | List of payments made for the consultation                         |
| `creator`           | User who created the consultation                                  |
| `doctor`            | Doctor Assigned to the consultation                                |
| `left`              | List of Users who have left the consultation                       |
| `logs`              | Consultation status log                                            |
| `patients`          | Users in the consultation who have the `patient` role              |
| `pendingPayments`   | List of payments pending for the consultation                      |
| `people`            | List of all the users scheduled to take part in the consultation   |
| `present`           | List of all the users preent for the consultation                  |
| `referral`          | Hospital referral attached to the consultation                     |
| `status`            | Current Status of the consultation.                                |
| `twilioRoom`        | Details of room created on Twilio for the video consultation       |
